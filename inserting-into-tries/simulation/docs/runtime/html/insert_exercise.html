1 Features of the artefact

  • This aretefact implements the practice for searching in Tries
  • It generates a trie and a word and asks the user to select the point of insertion and what's inserted
  • As the user clicks the nodes they get highlighted
  • The user gets feedback on his answer, by seeing the color of the nodes
  • Next, a new question will start off

2.2 Instruction Box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction-box">
  <button class="collapsible">Instructions</button>
  <div class="content">
    <ul>
      <li>You can see a linked list of a word and a Trie with a few
        words in it.</li>
      <li>Click on the <b>node in the TRIE</b> where you would start
        inserting this word from.</li>
      <li>Click on the first <b>letter of the WORD</b> which would be
        inserted in.</li>
      <li>NOTE: In the resultant Trie, the two nodes you clicked will
        be joined together.</li>
      <li>If you are right, both the nodes will be colored <b>GREEN</b>.
        Otherwise they will be RED.</li>
      <li>Click TRY NEXT WORD for starting on the next question,
        repeat a few times till comfortable.</li>
    </ul> 
  </div>
</div>

  insertExercise.reset();
});
</script>
