1 Features of the artefact

  • Artefact provides demo for insertion and search in suffix tries
  • User can feed a word and the suffix trie for it will be created.
  • User type a word and hit search, it will be searched for, coloring nodes on the way.
  • User can hit "Reset searches" to color all nodes back to original.
  • User can hit "Reset inserts" to delete all the words in the trie.

2.6.2 Declaration of the variables that initialize the trie

The Suffix Trie class is instantiated and binding all the click events on the HTML buttons to apropriate actions. 

<script type="text/javascript">
var suffix = new SuffixTrie("#render-5");
$('#insertion-suffix-submit').click(function () {
    suffix.prepare($('#insertion-suffix-text').val());
});
$('#search-suffix-submit').click(function () {
    suffix.search($('#search-suffix-text').val());
});
$('#reset-search-suffix').click(function () {
    $('.node').removeClass('active');
});
$('#reset-insert-suffix').click(function () {
    suffix.graph.tree.reload();
});
</script>
