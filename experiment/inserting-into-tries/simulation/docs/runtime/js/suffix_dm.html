1 Demonstration artefact for Insertion and Search in Tries

1.1 Features of the artefact

  • Inherits all features of Tries
  • Overrides: Insertion of a new Word by not putting in a $ at the end of word - insert(word)
  • Allows for: Prepare the whole suffix trie for a single word - prepare(word)

2 JavaScript Code

2.1 Class constructor

Creates a new suffix Trie, same as a regular Trie, given a container HTML element. 

class SuffixTrie extends Trie {

  constructor(container, animateDelay = 500) {
    super(container, animateDelay);
  }

2.2 Create for a word

Creates the suffix try for a word by iterating over all suffixes and inserting. 

  prepare(word) {
    this.graph.tree.reload();
    for (let i = 0; i < word.length; i++) {
      setTimeout(function (graph) { graph.insert(word.substring(i)) }, this.delay * i, this);
    }
  }

2.3 Override insert

Override insert to remove the trailing $ so that implicit endchar is a possibility. 

  insert(word) {
    word = word.toUpperCase();
    let data = this.graph.tree.getNodeDb()["db"];
    this._insert(word, data[0], data);
  }
}
